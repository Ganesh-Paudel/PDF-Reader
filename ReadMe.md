# PDF Reader

This application is supposed to make reading a pdf much more easy and accessible. Default PDf reader are not that good and the advanced one takes **money** .

# Application 

- Able to access and display pdf,
- Ability to zoom in and out
- Ability to highlight sentences. 
- Ability to have **Reader Mode **

## Reader Mode,

User will be able to choose the options, The user can have choices like going sentence by sentence with highlighting or paragraph by paragraph and so on. This way the whole page is not in display. It shows step by step. This will be modifiable so user can change based on their preference.
